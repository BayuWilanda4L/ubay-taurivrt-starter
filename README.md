## Getting Started
## Installation

```
git clone https://github.com/BayuWilanda4L/ubay-taurivrt-starter
```

Install the dependencies and devDependencies and start the server.

```sh
cd ubay-taurivrt-starter

yarn
#or with npm
npm i

#start the server
yarn tauri dev
```
